Laravel 8

GitHub

## Процедура развертывания

1. Открыть терминал и ввести "composer install"
2. Создать базу данных MySql
3. Открыть файл .env и настроить MySql DB_CONNECTION
4. Открыть терминал и ввести "php artisan migrate"
